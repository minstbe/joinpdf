    How to Use

    Split PDF

    Select a PDF to see page thumbnails. Click pages to select/deselect, or shift+click for range.

    Drag selected pages in the bottom strip to reorder them.

    Reverse ranges are supported (e.g., 7-2). Duplicates keep order.

    Optionally set an output filename. Click Extract Pages to download.

    Merge PDFs

    Select multiple PDFs. Drag items or use ▲/▼ to reorder.

    Optionally set an output filename. Click Merge Files to download.

    Privacy

    No upload. Files never leave your device; processing happens locally.
